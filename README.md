# StudyFlow — фронтенд учебного трекера

**StudyFlow** — это веб-приложение для трекинга учебных целей, которое помогает студентам повышать эффективность планирования учебного процесса.  
Фронтенд-часть реализует пользовательский интерфейс для взаимодействия с системой целей, расписанием и аналитикой.

---

## Архитектура

Фронтенд состоит из отдельных HTML-страниц, каждая из которых отвечает за определённую часть пользовательского интерфейса.

### Иерархия экранов
index.html — главная страница

├── Targets.html — цели и задачи

├── Calendar.html — календарь с целями

├── Achievments.html — достижения

├── Analitics.html — аналитика прогресса

└── Profile.html — профиль пользователя


---

## Технологии

- **HTML5** — структура страниц  
- **CSS3** — стилизация интерфейса  
- **JavaScript (ES6)** — логика взаимодействия и навигации  
- **GitHub Pages** — хостинг проекта
